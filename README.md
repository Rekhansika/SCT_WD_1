# Responsive Landing Page
This project is a responsive and interactive landing page developed as part of my internship task at **SkillCraft Technology**.

## 📌 Task Description
The objective of this project is to create an interactive navigation menu that:
- Remains fixed and visible while scrolling
- Changes its color and style when the page is scrolled
- Includes hover effects on navigation menu items
- Provides smooth navigation between different sections
- Works responsively across desktop, tablet, and mobile devices

## ✨ Features
- Fixed Navigation Bar
- Interactive Navbar
- Navbar Style Change on Scroll
- Navigation Menu Hover Effects
- Active Navigation Link Highlighting
- Smooth Scrolling
- Responsive Design
- Mobile Hamburger Menu
- Multiple Landing Page Sections

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 🚀 How to Run the Project
Clone the repository.
Open the project folder in Visual Studio Code.
Open index.html in your browser or use the Live Server extension.
Scroll through the page to see the interactive navigation bar effects.

## 📱 Responsive Design
The website is designed to work across different screen sizes, including:
Desktop
Tablet
Mobile
On smaller screens, the navigation menu automatically changes into a hamburger menu.
